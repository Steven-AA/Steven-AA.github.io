  • 析构函数 Destructors

  • Garbage Collection @q

    • Finalize()
      • 由系统自动调用
      • 时间不定
      • 最好不要 override
    • Dispose()
      • 手动继承 IDisposable 接口
      • 手动实现 Dispose 清理托管资源
      • 应该在Dispose()中调用GC.SuppressFinalize(this)通知垃圾回收器，此对象已经不再需要执行析构函数以免重复执行。但如果用户忘记了调用Dispose()，则垃圾回收器仍然会执行析构函数，保证非托管资源会被清除
